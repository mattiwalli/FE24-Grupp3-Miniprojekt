API Information

I chose the CurrencyLayer API for my project because it was easy to use. I only needed an API key to get started. This API lets me convert one currency to another and also shows a list of all available currencies. The documentation was clear and simple. I would recommend this API for small and medium projects since it is easy to access and works well.
